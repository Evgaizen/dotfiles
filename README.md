# Dotfiles

## Installation

### zsh
```sh
ln -s ~/dotfiles/zsh/.zshrc ~/.zshrc
source ~/.zshrc
```

### Neovim
```sh
ln -s ~/dotfiles/nvim ~/.config/nvim
```

### Alacritty
```sh
ln -s ~/dotfiles/alacritty ~/.config/alacritty
```

#### Font installation
```sh
brew install homebrew/cask-fonts/font-jetbrains-mono-nerd-font
```

### tmux
```sh
ln -s ~/dotfiles/tmux/.tmux.conf ~/.tmux.conf
```
